# پرودید — فروشگاه اینترنتی 🥩

وب‌سایت فارسی فروشگاه **پرودید** (پروتئین هر خانه) — فروشگاه تخصصی مواد پروتئینی و فودشاپ در کاشان.

- 📍 کاشان، خیابان امیرکبیر، روبه‌روی بیمارستان متینی
- 📞 ۰۹۱۳۰۶۱۲۰۱۹
- 📷 اینستاگرام: [@PRODID1](https://www.instagram.com/prodid1/)

## امکانات سایت

- کاملا **فارسی و راست‌چین (RTL)** با فونت وزیرمتن (به‌صورت لوکال)
- طراحی **Mobile First** با منوی پایین مخصوص موبایل
- فروشگاه با **جست‌وجو، فیلتر دسته‌بندی، فیلتر نوع فروش (وزنی/عددی) و مرتب‌سازی**
- **سبد خرید** با ذخیره‌سازی در مرورگر (localStorage)
- پشتیبانی از **محصولات وزنی** (قیمت هر کیلو + پیام شفاف «مبلغ نهایی پس از وزن‌کشی»)
- **تسویه حساب** با انتخاب روز و بازه زمانی تحویل، روش پرداخت و توضیحات سفارش
- ثبت نهایی سفارش از طریق **واتس‌اپ** (بدون نیاز به سرور) + گزینه سفارش تلفنی
- **حساب کاربری ساده** (ذخیره مشخصات و تاریخچه سفارش روی دستگاه کاربر)
- صفحات: اصلی، فروشگاه، محصول، سبد، تسویه، حساب، درباره ما، تماس، ارسال و سفارش، سوالات متداول، قوانین

## ساختار پروژه

```
index.html              صفحه اصلی
pages/                  بقیه صفحات (shop, product, cart, checkout, account, ...)
assets/css/styles.css   استایل کامل سایت
assets/js/data.js       اطلاعات برند + دسته‌بندی‌ها + محصولات و قیمت‌ها
assets/js/app.js        منطق سایت (سبد خرید، رندر صفحات، ثبت سفارش)
assets/fonts/           فونت وزیرمتن
assets/img/products/    عکس واقعی محصولات (اختیاری — پایین را بخوانید)
```

سایت کاملا **استاتیک** است؛ نیازی به بیلد یا سرور ندارد و روی GitHub Pages یا هر هاست ساده‌ای اجرا می‌شود.

## ویرایش محصولات و قیمت‌ها

همه محصولات در فایل [`assets/js/data.js`](assets/js/data.js) تعریف شده‌اند:

```js
{ id: "kabab-koobideh", name: "کباب کوبیده", cat: "kebab",
  price: 830000, sale: "w", emoji: "🍢", desc: "..." }
```

- `price`: قیمت به تومان (برای محصولات وزنی، قیمت هر کیلو). مقدار `null` یعنی «استعلام قیمت».
- `sale`: `"w"` وزنی / `"u"` عددی-بسته‌ای
- `available: false` محصول را «ناموجود» می‌کند.
- `badge`: برچسب دلخواه مثل «پرفروش» یا «ویژه»

## افزودن عکس واقعی محصولات

برای هر محصول کافی است عکسی با نام `<id>.jpg` در پوشه `assets/img/products/` قرار دهید
(مثلا `kabab-koobideh.jpg`). عکس به‌صورت خودکار جایگزین آیکن پیش‌فرض می‌شود؛ نیازی به تغییر کد نیست.

**عکس دسته‌بندی‌ها:** عکس هر دسته را با نام `<id>.jpg` در پوشه `assets/img/categories/`
بگذارید (مثلا `meat.jpg` برای گوشت و استیک). نام‌ها: meat، kebab، chicken، coldcuts،
burger، veg، salad، fresh، dairy، market.

**عکس هیرو:** اگر فایلی با نام `assets/img/hero.jpg` اضافه کنید، به‌جای طرح گرافیکی
پیش‌فرض در هیروی صفحه اصلی نمایش داده می‌شود (بهترین ابعاد: حدود ۱۲۰۰×۸۰۰، عکس تیره و اشتهابرانگیز).

## انتشار روی GitHub Pages

1. در تنظیمات ریپو: **Settings → Pages**
2. Source را روی برنچ اصلی (`main`) و پوشه `/ (root)` بگذارید.
3. سایت روی `https://<username>.github.io/prodid/` در دسترس خواهد بود.

## قدم‌های بعدی پیشنهادی

- اتصال درگاه پرداخت آنلاین (مثل زرین‌پال) — نیازمند سرور/سرویس بک‌اند
- اتصال دامنه اختصاصی (مثلا `prodid.ir`)
- افزودن عکس‌های واقعی محصولات از اینستاگرام فروشگاه
- تکمیل دسته‌بندی‌های «به‌زودی» (پروتئینی تازه، لبنیات، اقلام سوپرمارکتی)
